# QAmatomer

Q&A のツリー構造で知識を整理するブラウザ完結型のメモツールです。

## 特徴

- **ツリー構造で整理** — 質問と回答を親子関係で階層化し、複雑な知識体系をわかりやすく管理できます
- **テキスト注釈** — 回答文中の任意のテキストを選択してハイライト・補足メモを追加できます。注釈から派生した子ノードも作れます
- **Markdown + 数式** — 回答は Markdown で記述でき、`$...$` / `$$...$$` で KaTeX 数式をレンダリングできます
- **全文検索** — `Ctrl+K` / `Cmd+K` でプロジェクト内の質問・回答・注釈を横断検索できます
- **ローカル完結** — データは `localStorage` に保存されます。サーバー不要でオフラインでも使えます（数式・アイコンは CDN を使用）
- **エクスポート / インポート** — JSON 形式でバックアップ・別環境への移行ができます

## 使い方

用途に応じて以下をブラウザで開くだけで動きます。インストール不要です。

```
安定運用: QAmatomer_stable.html
新機能追加・実験: QAmatomer_dev.html
```

## ファイルの役割

- `QAmatomer_stable.html` — 現在の安定運用版（既存利用向け）
- `QAmatomer_dev.html` — 新機能追加・検証用

運用の混乱を避けるため、機能実装後は `dev` で確認してから `stable` へ反映してください。

## データ構造

| 概念 | 説明 |
|---|---|
| **プロジェクト** | まとめのひとまとまり。複数作成できます |
| **ノード** | 質問（タイトル）と複数のセクション（回答・解説）で構成される単位 |
| **セクション** | ノード内の回答ブロック。タイトルと Markdown 本文を持ちます |
| **注釈** | セクション内のテキストに付けるハイライト＋補足メモ |

## キーボードショートカット

| キー | 動作 |
|---|---|
| `Ctrl+K` / `Cmd+K` | 全文検索を開く / 閉じる |
| `Escape` | 検索・選択メニューを閉じる |

## エクスポート形式

```json
{
  "type": "annotator-qa-export",
  "version": 2,
  "exportedAt": 1234567890,
  "projects": [
    {
      "name": "プロジェクト名",
      "nodes": { ... },
      "annotations": [ ... ]
    }
  ]
}
```

## 技術スタック

- React 18 (CDN)
- Tailwind CSS (CDN)
- Marked — Markdown パーサー
- KaTeX — 数式レンダリング
- Phosphor Icons

データの保存先: `localStorage`（キー: `annotator-qa-project-list`, `annotator-qa-project-{id}`）
