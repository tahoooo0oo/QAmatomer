# CHANGELOG

## QAmatomer v3（現在の安定版）

**主な変更**
- ノードの上下並び替えボタン（ツリー・フラットリスト両方）を追加
- 「段落を追加」時に新しい段落へ自動スクロール
- 注釈・子要素リストをデフォルト折りたたみに変更（ルートは展開）
- 段落タイトルへの Markdown・数式レンダリング対応
- 段落編集ジャンプボタン（閲覧ペインから直接その段落の編集位置へジャンプ）

**削除**
- ノード間リンク機能（複雑さに対してメリットが薄いため廃止）
- スナップショット HTML 生成機能（同上）

---

## QAmatomer v2

**追加**
- 全文検索（`Ctrl+K` / `Cmd+K`）— 質問・セクション・注釈を横断検索
- ノード間リンク機能（v3で廃止）

---

## QAmatomer v1（初期版）

LLMmatomer から Q&A ツリー専用ツールとして分離して作成。

**主な機能**
- Q&A ツリー構造（階層的なノード管理）
- テキスト選択による注釈・ハイライト
- 注釈から派生した子ノードの作成（多重階層）
- 再帰的なツリーナビゲーター（左ペイン）
- セクション形式の回答エディタ（タイトル＋本文の複数ブロック）
- ツリーの全展開・全折りたたみボタン
- 複数プロジェクト管理・JSON エクスポート/インポート

---

## LLMmatomer（前身ツール）

QAmatomer の元になった PDF テキスト注釈ツール。以下のバージョンが archive に保存されている。

| バージョン | 主な内容 |
|---|---|
| v1 | PDF テキスト選択注釈の基本実装 |
| v2 | KaTeX 数式対応の注釈、ブレッドクラムトレイル |
| v3 | 注釈ツリービュー、テキスト編集 UX 改善 |
| v4 | ダブルクリックでエディタの対応行へジャンプ |
| v5 | アウトラインナビゲーション（`#` / `##` 見出しベース）|
| lineAnnotation_experimental | 行ベースアンカーの実験的実装 |
